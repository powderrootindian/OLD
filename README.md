## 🌿 The Vision
**Powder Root** is built for the modern minimalist. Moving away from cluttered, traditional e-commerce layouts, this project focuses on a "Pure & Raw" editorial aesthetic. It features a cinematic video hero section and a high-contrast dark theme designed to elevate the brand's visual identity.

## 🚀 Key Features
* **Editorial UI:** Clean, immersive design using **Cinzel** and **Montserrat** typography.
* **Google Authentication:** Secure user identity management via Firebase Auth.
* **Multi-Step Linear Checkout:** A guided, logic-based flow designed for conversion:
    1.  **Item Verification:** Real-time quantity management in a side-drawer bag.
    2.  **Shipping Entry:** Dedicated step for address collection.
    3.  **UPI Payment:** Integrated QR code for direct manual payments.
    4.  **WhatsApp Order Sync:** Automated formatting of order details for direct business communication.
* **Cloud Persistence:** Integration with **Firestore** to record confirmed orders and customer data.

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3 (Custom Flexbox & Grid), Vanilla JavaScript (ES6+).
* **Database & Auth:** Firebase (Authentication & Firestore).
* **Icons & Fonts:** FontAwesome 6.5.1, Google Fonts.
* **Communication:** WhatsApp Business API (URL schemes).

## 📦 Deployment
The application is currently optimized for deployment on **GitHub Pages** or **Firebase Hosting**.

### Installation
1.  Clone the repository:
    ```bash
    git clone [https://github.com/your-username/powder-root.git](https://github.com/your-username/powder-root.git)
    ```
2.  Open `index.html` via a local server (like VS Code Live Server).
3.  Ensure your `firebaseConfig` in `js/app.js` is correctly linked to your Firebase Project.
